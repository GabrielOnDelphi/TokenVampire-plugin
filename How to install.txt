ClaudeTokenVampire — Installation
=========================

Install:
  1. Copy this folder somewhere permanent (e.g. C:\Tools\ClaudeTokenVampire)
  2. Double-click Install.cmd
  3. In Claude Code, run:  /reload-plugins

Launch (two ways):
  - Fast:  Type "launch vampire" or "token monitor" in Claude Code (instant, no thinking delay)
  - Skill: Type /claudetokenvampire:monitor in Claude Code (~5 sec, loads full context)



____________________




Uninstall:
  1. Double-click Uninstall.cmd
  2. In Claude Code, run:  /reload-plugins

Troubleshooting:
  - "Plugin not found in marketplace" — run Install.cmd again, then /reload-plugins
  - Skill doesn't appear — run /reload-plugins (skills are cached at reload time)
  - App won't launch — check that bin\ClaudeTokenVampire.exe exists in this folder
